## 游戏概述

### 概念

玩家扮演**青少年的大脑**，与身体各器官（拟人化角色）建立羁绊。每个器官拥有两个独立属性：**先天健康值**（代表身体机能）和**好感度**（代表情感连接）。游戏时间跨度为高中三年，玩家需要在学业压力、意外事件和日常选择中平衡健康、情感与心理，最终达成不同结局。

### 核心理念

- **天命难违，尽力而为**：健康值受遗传、意外影响，大脑只能通过良好习惯和情感羁绊去延缓损失，无法完全掌控。
- **情感的力量**：高好感度能为器官提供保护，减少伤害，体现"爱能带来生命力"。
- **心理健康同等重要**：大脑本身的娱乐值（心理健康）过低会导致游戏失败。

### 核心体验

- 拟人化器官陪伴，温暖治愈
- 策略性养成，在有限行动中平衡多目标
- 真实高中生活代入感（考试、压力、成长）
- 随机事件带来的不可预测性与重玩价值

## 核心玩法

### 游戏循环

```
每月开始 → 分配4个行动点（月度决策） → 触发随机事件（可选） → 月末结算衰减 → 进入下月
```

### 胜利/失败条件

**失败条件**：
- 任一器官健康值 ≤ 0 且未触发"善意谎言"机制 → 游戏结束
- 器官陷入昏迷后三回合内未恢复至 ≥10 → 游戏结束
- 娱乐值归零后进入"抑郁期"，两周内未提升至 ≥20 → 游戏结束

**胜利条件**：完成高中三年（36个月）后，根据综合评价给出结局。

### 资源系统

| 资源 | 全称 | 范围 | 初始值 | 说明 |
|------|------|------|--------|------|
| Health | 器官健康值 | 0-100 | 45-65 | 归零则器官衰竭 |
| Affection | 器官好感度 | 0-150 | 30 | 高好感可减免伤害 |
| Entertainment | 大脑娱乐值 | 0-100 | 60 | 代表心理健康 |
| Stress | 压力值 | 0-200 | 30 | 过高会加速健康衰减 |
| Academic | 学业值 | 0-100 | 40 | 代表学习进度与成绩 |

## 数值系统

### 健康值（Health）

- 上限：100
- 恢复规则：高一高二可恢复至初始值+20，高三仅能延缓衰减
- 危急状态：≤15时触发紧急救治事件
- 善意谎言机制：好感度≥90的器官健康归零后不会立即结束，进入昏迷状态

### 好感度（Affection）

- 上限：150（高一100，高二120，高三150）
- 作用：减免健康衰减和意外事件伤害
- 分段保护：0-50无保护，50-100每点减免0.5%，100-150每点减免1%
- 好感阈值（30/60/90/120/150）触发个人剧情事件

### 大脑娱乐值（Entertainment）

- 定义：心理健康程度
- <30：行动效果减半，每3天触发抑郁强制休息
- >80：行动效果+20%，有概率获得额外行动点
- 情绪危机：低于20时触发特殊选择事件

### 压力值（Stress）

- >120：每月额外增加每个器官健康衰减2-4点
- >180：触发心理危机事件

### 学业值（Academic）

- 每月学习行动增加5~10点（随机浮动）
- 考试事件根据学业值决定成绩等级
- 结局判定：≥70为学业良好，≥90为学业极高

## 时间与行动系统

### 时间结构

- 总跨度：高中三年（高一上 → 高三下）
- 每个学期：6个月（6个大回合）
- 每月结构：4个行动点 + 月末自动结算 + 周末特殊事件
- 快速模式：可跳过高一高二，直接进入高三冲刺

### 行动类型

| 行动 | 主要效果 |
|------|----------|
| 运动 | 心脏健康+，足健康+，娱乐+，压力- |
| 按时吃饭 | 胃健康+，胃好感+ |
| 学习 | 学业+，压力+，器官健康- |
| 娱乐 | 娱乐+，压力-，眼健康- |
| 与角色互动 | 对应器官健康+，好感+ |
| 早睡 | 各器官健康+，肝健康+，压力- |

> 行动效果随机浮动（80%-120%）

## 意外事件系统

### 事件分类

- 学业相关：月考失利、竞选落选
- 健康危机：流感、运动拉伤、视力下降
- 家庭/社交：父母争吵、朋友矛盾
- 环境因素：雾霾、食堂食物中毒
- 遗传/先天：心咲心悸、肺清哮喘

### 事件结构

每个事件包含描述文本、2-3个选项及对应数值变化后果。

## 结局与评价

### 结局类型

| 结局名称 | 条件 |
|----------|------|
| **绽放·无悔青春** | 所有器官健康≥60，好感≥100，娱乐≥70，学业≥70 |
| **情深负重** | 平均好感≥130，平均健康<40 |
| **乐天派** | 娱乐>85，平均健康>70，平均好感<60 |
| **书呆子** | 学业≥90，平均健康<30或娱乐<20 |
| **遗憾退场** | 任一器官健康归零且未触发善意谎言 |
| **抑郁沉沦** | 娱乐值归零后抑郁期未恢复 |
| **平凡之路** | 各项指标中等 |

### 综合评分公式

```
总分 = 平均健康值×0.3 + (平均好感度/150)×35 + 最终娱乐值×0.2 + (最终学业值/100)×15
```

（满分100分）
